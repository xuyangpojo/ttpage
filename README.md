# TTPage 双列视频推荐 安卓客户端
可在 Android 系统上运行的视频推荐应用，采用 MVVM 架构，支持双列布局、视频播放、视频评论区等功能。实现了类似抖音的双列外流与视频内流页面。

本系统作为 字节跳动工程训练营-客户端-《推荐双列页面》课题作业。

开发人：北京航空航天大学 27届 软件工程专业(本科) 徐洋

联系邮箱: xuyangpojo@icloud.com

### 指令手册
`./gradlew test` 运行单元测试 

`./gradlew connectedAndroidTest` 运行集成测试
