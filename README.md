# 双列推荐系统

### Feature/Basic 12.9
1. 首页双列推荐页面，屏幕分成左右两部分，从上到下显示内容卡片。
2. 实体: 暂时使用纯文字作为一个内容。包括作者，发布时间，点赞数，评论数，是否是热门内容，标题。
