# 双列推荐系统
注意: 使用MVVM架构
### Feature/Basic
1. 首页双列推荐页面，屏幕分成左右两部分，从上到下显示内容卡片。
2. 实体: 暂时使用纯文字作为一个内容。包括作者，发布时间，点赞数，评论数，是否是热门内容，标题。
3. 点击内容跳转到内容详细页。暂时只显示实体字符串信息。
4. 在内容详细页滑动返回主页页面

### Feature/Video
1. 实体增加视频封面和视频文件的属性。
2. 增加

### Feature/Comment
1. 评论区功能
2. 转发在剪切板复制链接的功能
3. 点赞和账户评论、回复的功能。

### Feature/Star
1. 封面根据不同的比例自适应布局
2. 首页页面的顶部存在不同话题，可左右滑动切换
3. 增加点击和退出详细页时，封面放大，视频缩小的详细细节
4. 上下拉动刷新与加载
5. 内部上下切换
