# 本地资源使用说明

## 📁 资源文件放置位置

### 1. 视频文件（MP4）
**位置**: `app/src/main/res/raw/`

**命名规则**:
- 文件名必须是小写字母、数字和下划线
- 不能包含空格和特殊字符
- 示例：`video1.mp4`, `sample_video.mp4`

**示例文件结构**:
```
app/src/main/res/raw/
├── video1.mp4      # 第一个视频
├── video2.mp4      # 第二个视频
├── video5.mp4      # 第五个视频
└── video7.mp4      # 第七个视频
```

### 2. 图片文件（PNG/JPG）
**位置**: `app/src/main/res/drawable/`

**命名规则**:
- 文件名必须是小写字母、数字和下划线
- 不能包含空格和特殊字符
- 支持格式：`.png`, `.jpg`, `.jpeg`, `.webp`
- 示例：`cover1.png`, `video_cover.png`

**示例文件结构**:
```
app/src/main/res/drawable/
├── cover1.png      # 第一个视频封面
├── cover2.png      # 第二个视频封面
├── cover5.png      # 第五个视频封面
└── cover7.png      # 第七个视频封面
```

## 🔧 如何添加资源文件

### 方法1：通过Android Studio
1. 在项目视图中，右键点击 `app/src/main/res/` 目录
2. 选择 `New` → `Android Resource Directory`
3. 选择资源类型（`raw` 或 `drawable`）
4. 将文件拖拽到对应目录中

### 方法2：直接复制文件
1. 在文件管理器中找到 `app/src/main/res/raw/` 或 `app/src/main/res/drawable/` 目录
2. 直接复制你的 `.mp4` 或 `.png` 文件到对应目录
3. 确保文件名符合命名规则

## 📝 代码中的使用方式

### Repository中的配置
在 `ContentRepository.kt` 中，使用资源名称（不含扩展名）：

```kotlin
Content(
    id = "1",
    title = "...",
    // ...
    videoCover = "cover1",  // drawable资源名称（不含.png）
    videoUrl = "video1"     // raw资源名称（不含.mp4）
)
```

### 资源名称映射
- **视频文件**: `video1.mp4` → 代码中使用 `"video1"`
- **封面图片**: `cover1.png` → 代码中使用 `"cover1"`

## ⚠️ 注意事项

1. **文件大小限制**: 
   - APK文件大小有限制，建议视频文件不要太大（< 50MB）
   - 如果视频很大，考虑使用网络URL或压缩视频

2. **资源ID生成**:
   - Android会自动为资源文件生成ID
   - 文件名就是资源ID（不含扩展名）

3. **重新构建**:
   - 添加新资源后，需要重新构建项目（Build → Rebuild Project）
   - Android Studio会自动识别新资源

4. **资源命名冲突**:
   - 确保不同资源类型的文件名不重复
   - 例如：`raw/video1.mp4` 和 `drawable/video1.png` 可以共存

## 🎯 示例配置

假设你有以下文件：
- `app/src/main/res/raw/video1.mp4`
- `app/src/main/res/drawable/cover1.png`

在 `ContentRepository.kt` 中这样使用：

```kotlin
Content(
    id = "1",
    title = "示例视频",
    author = "作者",
    publishTime = "1小时前",
    likeCount = 100,
    commentCount = 20,
    isHot = true,
    videoCover = "cover1",  // 对应 drawable/cover1.png
    videoUrl = "video1"      // 对应 raw/video1.mp4
)
```

## 🔍 验证资源是否正确添加

1. 在Android Studio中，打开 `app/src/main/res/raw/` 或 `drawable/` 目录
2. 确认文件显示在目录中
3. 构建项目，如果没有错误说明资源已正确添加
4. 运行应用测试视频播放和图片显示
