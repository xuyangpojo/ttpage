# 用户头像使用说明

## 📁 头像文件放置位置

**路径**: `app/src/main/res/drawable/`

## 📝 命名规则

头像文件名必须遵循以下规则：
- 格式：`avatar_{authorId}.png`
- `authorId` 是评论数据模型中的用户ID字段
- 文件名必须是小写字母、数字和下划线
- 不能包含空格和特殊字符

## 📋 使用示例

假设评论数据中的 `authorId` 为 `u101`，则对应的头像文件名应为：
```
avatar_u101.png
```

其他示例：
- 用户ID: `u102` → 头像文件: `avatar_u102.png`
- 用户ID: `u103` → 头像文件: `avatar_u103.png`
- 用户ID: `user_001` → 头像文件: `avatar_user_001.png`

## 🔧 如何添加头像文件

### 方法1：通过Android Studio
1. 在项目视图中，右键点击 `app/src/main/res/drawable/` 目录
2. 选择 `New` → `Image Asset` 或直接复制PNG文件
3. 将文件命名为 `avatar_{authorId}.png` 格式
4. 确保文件名符合命名规则

### 方法2：直接复制文件
1. 在文件管理器中找到 `app/src/main/res/drawable/` 目录
2. 直接复制你的PNG头像文件到该目录
3. 重命名为 `avatar_{authorId}.png` 格式

## 🎨 头像显示规则

1. **有头像文件时**：显示用户自定义头像（圆形裁剪）
2. **无头像文件时**：显示默认头像（Person图标，灰色背景）

## ⚠️ 注意事项

- 头像文件必须是PNG格式
- 文件名必须严格按照 `avatar_{authorId}.png` 格式
- 如果找不到对应的头像文件，系统会自动显示默认头像
- 头像会自动裁剪为圆形显示
- 建议头像尺寸为正方形（如 256x256 或 512x512 像素）

## 📍 代码中的使用

头像组件会自动根据 `authorId` 查找对应的头像文件：

```kotlin
UserAvatar(
    authorId = comment.authorId,  // 例如: "u101"
    modifier = Modifier.size(32.dp)
)
```

系统会自动查找 `avatar_u101.png` 文件，如果找不到则显示默认头像。
